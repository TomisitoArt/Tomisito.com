  • Dark Mode

    Mon, 26 May 2025 14:17 GMT

    Thanks to a request from Adam Phoebe, who was "flashbanged" by my site while browsing late at night, I've added a dark mode to the website. It relies on a CSS media query which detects your browser default preference, so I didn't need to add any JS or cookies.

    Thankfully, my site styles were built with CSS variables, so adding this new recolour was actually pretty straightforward!

  • GBJam12 finished!

    Sun, 22 September 2024 20:00 GMT
